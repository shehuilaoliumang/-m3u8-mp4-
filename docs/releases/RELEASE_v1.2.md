# v1.2 - 性能增强与交互优化

## 核心更新

### 1) UI 性能增强

- **进度刷新节流**：转换期间将高频进度回调合并为约 15fps 刷新，减少主线程事件风暴。
- **日志批量渲染**：短时间日志写入合并刷新，降低 `Text` 组件频繁重绘开销。
- **日志增量显示**：在默认筛选视图（全部/全部任务）下，日志只追加新行，不再每次全量重绘。
- **启动分阶段初始化**：拖放、依赖状态、FFmpeg 检测按阶段异步执行，降低启动黑屏等待。

### 2) 帮助中心体验优化

- **异步加载 README**：帮助窗口先展示容器，再后台读取解析文档，避免打开时卡顿。
- **缓存预热机制**：启动后后台预热帮助内容缓存，首次打开帮助速度显著提升。

### 3) 报错体验优化

- **只显示真实错误**：过滤 ffmpeg/ffplay 版本信息、流信息等常规输出，仅保留关键错误行。

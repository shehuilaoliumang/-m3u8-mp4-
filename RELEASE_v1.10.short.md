# v1.10 更新速览

- 新增 `mov/avi` 输出，支持前缀/后缀/时间戳命名。
- 新增 m3u8 完整性校验与首分片预览（`ffplay` 不可用时自动生成 5 秒临时预览）。
- 主界面新增“诊断与预览工具”，可一键校验、预览、清理临时预览文件。
- 日志中心支持分级显示、按任务筛选、仅失败快捷筛选与当前筛选导出。
- 增强断点续传（任务级 + 分片校验续传记录）。
- 跨平台安装/打包增强：`winget`/`brew`/`apt|dnf|yum`，新增 macOS DMG 与 Linux DEB/RPM 脚本。

## 升级提示

- 建议在“设置”中确认：输出格式、命名规则、提示音、会话结束自动清理临时预览开关。
- 首次使用 Linux DEB/RPM 打包脚本前，请先安装 `fpm`。

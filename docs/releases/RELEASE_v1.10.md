# v1.10 - 功能增强与 UI 体验升级

## 版本亮点
本版本重点围绕可用性、可观测性、跨平台、批量性能全面升级，新增 m3u8 辅助工具、输出格式扩展、日志中心增强、断点续传能力强化，并优化主界面交互体验。

## 新增功能

### 1) 输出与命名能力升级
- 新增输出格式支持：`mp4` / `mov` / `avi`
- 批量命名规则增强：
  - 文件名前缀
  - 文件名后缀
  - 时间戳命名（`YYYYMMDD_HHMMSS`）

### 2) m3u8 辅助工具
- 新增完整性校验：
  - 检测加密状态与加密方式
  - 识别 KEY 信息
  - 检查本地分片缺失
- 新增首分片预览：
  - 优先 `ffplay` 直接预览
  - 若不可用，自动生成 5 秒临时 mp4 预览

### 3) 日志中心增强
- 日志按级别分类显示：信息 / 成功 / 警告 / 失败 / 调试
- 支持按任务筛选日志
- 错误日志标红并支持一键复制
- 支持“仅失败”快捷筛选
- 日志导出支持“当前筛选结果”导出

### 4) 断点续传增强
- 保留任务级续传能力
- 新增分片校验续传进度记录（本地场景减少重复校验）

## UI 与交互优化

- 主界面新增“诊断与预览工具”分组：
  - 完整性校验
  - 预览首片段
  - 清理临时预览
- 工具入口从设置菜单迁移到主界面，操作更直观
- 主界面支持右侧滚动滑轨：
  - 当窗口高度不足时自动显示
  - 支持滚轮上下浏览完整内容
- 新增提示音（可开关）：完成 / 失败 / 取消
- 新增会话结束自动清理临时预览文件（可开关）

## 跨平台与打包

### FFmpeg 自动部署（跨平台）
- Windows：`winget`
- macOS：`brew`
- Linux：`apt-get` / `dnf` / `yum`

### 新增打包脚本
- `scripts/build_mac_dmg.sh`
- `scripts/build_linux_deb.sh`
- `scripts/build_linux_rpm.sh`

### Windows 打包优化
- 调整 spec / 打包参数
- 排除无用开发依赖，减小包体

- 新增并扩展测试覆盖：
  - 输出格式
  - m3u8 工具链
  - UI 辅助逻辑
